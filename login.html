Phytozome

O Phytozome consiste em uma vase de dados internacional que reune os genomas de diferentes espécies vegetais.

Entre as espécies disponíveis estão: Vigna unguiculata, Phaseolus lunatus, Phaseolus vulgaris, Glycine max, Glycine soja.

Por meio dessa plataforma é possíveis realizar diferentes análises, como: análise de ortologia, análise de sintenia, comparação entre genomas e etc.

Nela também é possível baixar diferentes genomas e analisá-los.

SOPMA

Recentemente, um novo método chamado método de previsão auto-otimizado (SOPMA) foi descrito para melhorar a taxa de sucesso na previsão da estrutura secundária de proteínas.

Foram relatadas melhorias provocadas pela previsão de todas as sequências de um conjunto de proteínas alinhadas pertencentes à mesma família.

Este método SOPM melhorado (SOPMA) prevê corretamente 69,5% de aminoácidos para uma descrição de três estados da estrutura secundária (alfa-hélice, beta-folha e bobina)

em um banco de dados completo contendo 126 cadeias de não homólogos (menos de 25 % de identidade) proteínas

SWISS-model

Voltar

SWISS-MODEL é um servidor web de bioinformática estrutural dedicado à modelagem de homologia de estruturas de proteínas 3D.

A modelagem de homologia é atualmente o método mais preciso para gerar modelos de estrutura de proteínas tridimensionais confiáveis e é usado rotineiramente em muitas aplicações práticas

Os métodos de modelagem de homologia (ou comparativos) fazem uso de estruturas de proteínas experimentais ("modelos") para construir modelos para proteínas relacionadas à evolução ("alvos").

Hoje, o SWISS-MODEL consiste em três componentes fortemente integrados:

(1) O pipeline SWISS-MODEL – um conjunto de ferramentas de software e bancos de dados para modelagem automatizada de estrutura de proteínas

(2) O SWISS-MODEL Workspace – um workbench de usuário gráfico

3) O SWISS-MODEL Repository – um banco de dados continuamente atualizado de modelos de homologia para um conjunto de proteomas de organismos modelo de alto interesse biomédico
